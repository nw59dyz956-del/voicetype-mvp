# VoiceType MVP

Минимальный локальный аналог Superwhisper для macOS: зажали правый `Command`, сказали фразу, отпустили, текст вставился в активное окно.

## Установка

```bash
brew install portaudio
pip install -e .
```

## Запуск

```bash
python -m voicetype_mvp
```

Или двойным кликом по файлу:

```text
Start VoiceType.command
```

При запуске появится иконка в menu bar:

- `🎙 idle` — ожидание
- `🔴 recording` — идет запись
- `⏳ processing` — идет транскрипция

## Разрешения macOS

Выдайте разрешения в System Settings -> Privacy & Security:

- **Microphone** — для записи
- **Accessibility** — для эмуляции `Cmd+V`
- **Input Monitoring** — для глобального хоткея

При запуске VoiceType автоматически пытается вызвать системные запросы разрешений и открывает нужные разделы System Settings. В списке приложений обычно нужно включить Terminal, Python или приложение, из которого был запущен `Start VoiceType.command`.

## Модель

При первом запуске `faster-whisper` скачает модель `small` (~460 MB) в:

```text
~/Library/Application Support/VoiceTypeMVP/models/
```

Язык не фиксируется жестко: модель сама распознает смешанную русско-английскую речь. Русские слова обычно будут кириллицей, а английские бренды и названия вроде `Apple`, `Google`, `OpenAI`, `ChatGPT`, `YouTube` — латиницей.

## Использование

1. Откройте Notes или другое приложение с активным текстовым полем.
2. Зажмите правый `Command`, который находится справа от пробела.
3. Скажите: `Привет, это тестовая запись`.
4. Отпустите клавиши.
5. Через пару секунд текст будет вставлен в активное окно.

## Ограничения

Это намеренно маленький прототип: без истории, базы данных, LLM-постобработки, настроек, выбора языка или модели в рантайме.
